# 📦 CodeAlpha Stock Manager

## 📌 Description

A command-line stock management system built in Python to manage products, update inventory, and calculate total investment.

## 🚀 Features

* Add / delete / update products
* Display stock
* Calculate total investment
* User-friendly menu system

## ▶️ How to Run

```bash
python stock.py
```
